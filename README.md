## 📖 About Doubly Linked List

A **Doubly Linked List (DLL)** is a linear data structure where each node contains:

* Data
* Pointer to the next node (`next`)
* Pointer to the previous node (`prev`)

Unlike a singly linked list, a doubly linked list allows traversal in both forward and backward directions, making insertion and deletion operations more flexible.

## 📚 Operations

* Insert at Beginning
* Insert at End
* Delete a Node
* Search for an Element
* Forward Traversal
* Backward Traversal
